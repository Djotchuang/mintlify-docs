Vous pouvez retirer de l’argent de votre compte Fapshi à tout moment grâce à la fonctionnalité **Retraits**. Que vous utilisiez Fapshi pour votre entreprise ou à titre personnel, les retraits vous permettent de transférer vos fonds vers un compte mobile money ou bancaire.

<Tip>Les retraits sont gratuits et peuvent être effectués **24h/24 et 7j/7** — à tout moment, n’importe quel jour.</Tip>

## Comment effectuer un retrait

Suivez ces étapes pour initier un retrait depuis votre tableau de bord :

**Étape 1 :** Connectez-vous à votre tableau de bord et allez à **Soldes > Retraits**

**Étape 2 :** Cliquez sur le bouton **« Retirer de l’argent »** en haut à droite.

**Étape 3 :** Remplissez les informations requises :

- Sélectionnez la destination (MTN MoMo, Orange Money ou compte bancaire)
- Entrez le montant à retirer
- Entrez votre **code PIN Fapshi**
- Vous pouvez ajouter une note facultative pour référence
- Confirmez la transaction

<Note>
Pour effectuer un retrait, vous devez avoir défini un **code PIN Fapshi**.  
Si vous ne l’avez pas encore fait, consultez notre [guide pour définir votre code PIN](https://fapshi.com/fr/aide-et-support/how-to-set-your-fapshi-pin).  
Avez-vous oublié votre code PIN ? Découvrez [quoi faire ici](https://fapshi.com/fr/aide-et-support/forgot-pin).
</Note>

**Étape 4 :** Une fois la transaction confirmée, elle sera traitée et apparaîtra sur votre page **Retraits**. Votre **solde de compte sera mis à jour** en conséquence.

## Conditions

- Votre **solde principal** doit être **supérieur ou égal** au montant que vous souhaitez retirer.
- Vous devez avoir **défini votre code PIN Fapshi** pour valider les retraits.

## Notes

- ✅ Les retraits via MTN MoMo et Orange Money sont traités instantanément.
- 🏦 Les virements bancaires peuvent prendre jusqu’à **2 jours ouvrables**.
- 💡 Vous pouvez consulter l’historique complet de vos retraits dans le tableau de bord ou via l’application mobile.
