<Tip>Créer un compte sur Fapshi est **simple et GRATUIT**. Avec un compte, vous pouvez **accéder à TOUS les produits Fapshi** et commencer à recevoir des paiements.</Tip>

---

## Étapes

1. **Visitez la page d’inscription** : Allez sur [la page d’inscription](https://dashboard.fapshi.com/register) ou téléchargez l’application mobile pour [Android](https://play.google.com/store/apps/details?id=com.fapshi) ou [iOS](https://apps.apple.com/ng/app/fapshi/id6443700859) et remplissez les champs **nom d'utilisateur**, **e-mail**, **numéro de téléphone** et **mot de passe**.
   - Si vous utilisez l'application mobile, appuyez sur "Vous n'avez pas de compte ? Inscrivez-vous" pour vous enregistrer.
<Note>
    Voici quelques conseils lors de la création de votre compte :
    - Le nom d’utilisateur doit contenir entre 4 et 40 caractères. Il peut inclure des lettres, des chiffres, des tirets, des espaces et des underscores (_).
    - Tous les reçus de paiement et les informations importantes vous seront envoyés par e-mail. Veuillez donc fournir un e-mail valide, de préférence celui que vous utilisez le plus.
    - Nous vous recommandons d’utiliser votre numéro WhatsApp pour l’inscription.
    - Le mot de passe doit comporter au moins six caractères.
</Note>

2. **Vérifiez votre adresse e-mail** : Un e-mail contenant un code de vérification vous sera envoyé pour confirmer votre adresse. Copiez ce code et collez-le dans le champ de vérification. Une fois vérifié, vous serez redirigé vers la page de connexion.
   - Si vous ne recevez pas l’e-mail de vérification, vérifiez votre dossier **Spam**. Si vous utilisez un iPhone, consultez le dossier **Courrier indésirable**. Une fois le code copié, signalez l'e-mail comme **"non spam"** afin que les prochains e-mails de Fapshi arrivent dans votre boîte principale.
   - Si vous ne recevez toujours pas l’e-mail, vous pouvez demander un nouveau code en cliquant sur le bouton **"Renvoyer le code"** sur la page de vérification.

<Warning>
    **NE SAUTEZ PAS L’ÉTAPE DE VÉRIFICATION PAR E-MAIL !** Si vous ne vérifiez pas votre e-mail, vous ne pourrez pas vous connecter à votre compte.
</Warning>

---

Une fois votre compte créé, vous pouvez vous connecter au [Tableau de bord Fapshi](https://dashboard.fapshi.com).

<Danger>
Si après cinq tentatives vous ne parvenez toujours pas à vous connecter, vous devrez attendre une heure avant de réessayer. Utilisez l’option [**Mot de passe oublié**](https://dashboard.fapshi.com/forgot-password) si vous avez oublié votre mot de passe. Cela permet de prévenir les tentatives de phishing.
</Danger>
