<Tip>
  Vous pouvez tester l'API de Fapshi en mode test sans activer votre compte. Cela vous permet d’intégrer et de tester votre application avant de passer en mode réel. Toutefois, pour commencer à recevoir des paiements réels, vous devez activer votre compte en complétant le processus de vérification KYC (Know Your Customer).
</Tip>

---

## Étapes pour activer votre compte Fapshi

1. **Connectez-vous à votre tableau de bord Fapshi** : Accédez au [Tableau de bord Fapshi](https://dashboard.fapshi.com) et connectez-vous avec vos identifiants.
2. **Allez dans les paramètres du compte** : Cliquez sur le bouton **"Activer le compte"** ou sélectionnez **Conformité** dans le menu. Si vous utilisez l’application mobile, allez dans les paramètres puis appuyez sur **"Activer le compte"**.
3. **Complétez la vérification KYC** : Remplissez les informations requises pour la vérification KYC, y compris le téléchargement de documents d'identité et les informations relatives à votre entreprise.
4. **Soumettez pour révision** : Une fois le formulaire KYC rempli, soumettez-le pour validation. Vous recevrez une notification dès que votre compte sera activé.
<Note>L’activation du compte est généralement effectuée en moins de 24 heures. Si vous ne recevez aucun e-mail de confirmation passé ce délai, veuillez [contacter le support](https://wa.me/237658610472).</Note>
5. **Commencez à recevoir des paiements** : Après l’activation, vous pourrez commencer à recevoir des paiements en direct via Fapshi.
<Info>Nous vous recommandons de tester votre intégration en mode test avant de passer en production. Il peut également être nécessaire de vous déconnecter puis de vous reconnecter pour voir les changements.</Info>

## Informations supplémentaires

- Pour la pièce d'identité, nous n'acceptons que les formats **JPG, JPEG** et **PNG** ; assurez-vous que vos fichiers sont bien dans l’un de ces formats. Si ce n’est pas le cas, vous pouvez utiliser [Image Online Converter](https://image.online-convert.com/convert-to-png) pour les convertir. Assurez-vous aussi que les images sont suffisamment claires pour que les détails soient bien visibles.
- Si vous utilisez un récépissé de carte d’identité, assurez-vous de soumettre la partie où la validité a été prolongée (verso du récépissé).

Voici la liste des pièces valides (par ordre de préférence) :
- Carte nationale d'identité camerounaise
- Passeport camerounais
- Récépissé de carte d'identité camerounaise
<Warning>Le document soumis doit être valide au moment de l'envoi. AUCUN autre document ou pièce expirée ne sera accepté.</Warning>
- Le nom de votre entreprise est un identifiant unique. **Assurez-vous d’utiliser le nom exact car il ne pourra plus être modifié**. Ajoutez également une adresse physique traçable (idéalement avec un quartier et une ville), ainsi qu’un numéro de téléphone WhatsApp valide.

## Résolution des problèmes

<AccordionGroup>
  <Accordion title="Erreur : nom d'image rouge affiché">
    **Solution :** Cette erreur survient généralement lorsque l’image que vous avez téléchargée n’est pas dans un format pris en charge ou est trop volumineuse. Les formats acceptés sont JPG, JPEG et PNG. Un nom d’image en rouge signifie que l’image est invalide. Vérifiez les formats et la taille du fichier.
  </Accordion>

  <Accordion title="Problème : Email non reçu">
    **Solution :** Vérifiez vos courriers indésirables ou spam. Si vous ne trouvez toujours pas l’e-mail, [contactez le support](https://wa.me/237658610472).
  </Accordion>
</AccordionGroup>
